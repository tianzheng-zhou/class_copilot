# 听课助手 (class_copilot)

实时课堂辅助工具，帮助学生在课堂上快速获取高质量的答题参考。

## 功能

- **实时语音转写**：阿里云百炼 ASR，支持 Fun-ASR 和千问3-ASR 两种模型可切换
- **问题检测**：自动识别教师提出的问题
- **智能答案生成**：基于课堂上下文生成简洁版 + 展开版参考答案
- **主动问答**：课堂上实时向 AI 提问（使用 qwen3.5-plus）
- **课堂记录**：保存音频、转写及问答记录
- **中英双语**：支持英文授课自动翻译

## ASR 模型选择

可在设置中切换语音识别模型，重新开始会话后生效：

| 模型 | 特点 | 价格 |
|------|------|------|
| Fun-ASR（默认） | 课堂/演讲场景优化，中英日混合识别 | ~1.19 元/时 |
| 千问3-ASR | 多语种高精度，情感识别 | ~1.19 元/时 |

## 快速开始

```bash
# 方式一：双击启动
start.bat

# 方式二：命令行
pip install -e .
python run.py
```

首次运行需在设置中配置阿里云百炼 API Key（ASR 和 LLM 共用同一个 Key）。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+S` | 开始/停止监听 |
| `Ctrl+Shift+Q` | 手动触发提问检测 |
| `Ctrl+Shift+H` | 隐藏/显示悬浮窗 |
| `Ctrl+Shift+C` | 复制最新答案 |
| `Ctrl+Shift+T` | 切换简洁/展开版答案 |
| `Ctrl+Shift+A` | 呼出主动提问输入框 |
| `Ctrl+Shift+F` | 切换 LLM 输入模式 |

## 项目结构

```
src/
├── asr/            # 语音识别（Fun-ASR / Qwen3-ASR 双引擎）
├── config/         # 配置与常量
├── core/           # 核心业务（会话管理、转写、说话人）
├── llm/            # 大模型（问题检测、答案生成、翻译）
├── storage/        # 数据持久化（SQLite、音频文件）
├── ui/             # PyQt6 界面（悬浮窗、系统托盘）
└── utils/          # 工具（加密、通知）
```

## 技术栈

- Python 3.12 + PyQt6
- 阿里云百炼 DashScope SDK（ASR + LLM）
- ASR：Fun-ASR / 千问3-ASR（可切换）
- LLM：qwen3.5-flash（自动答案）+ qwen3.5-plus（主动问答）
- SQLite 本地存储
- Fernet 加密（API Key 安全存储）
