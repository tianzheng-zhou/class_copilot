# 听课助手 (Class Copilot)

实时听课辅助工具 — 课堂语音转写、问题检测与智能答案生成。

## 快速开始

```bash
# 安装依赖
pip install -e .

# 启动应用
python -m class_copilot
```

启动后自动打开浏览器访问 `http://localhost:8765`。

## 功能概览

- **实时语音转写**：支持中英文及混合语言
- **问题检测**：自动识别教师提问，弹窗提醒
- **智能答案生成**：简洁版+展开版双答案
- **主动提问**：实时向 AI 提问课堂相关内容
- **高精度精修**（可选）：课后/课中高精度二次转写
- **课堂记录**：完整的录音、转写、问答归档
- **系统托盘**：后台运行，全局快捷键操作

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Shift+S | 开始/停止监听 |
| Ctrl+Shift+Q | 手动触发问题检测 |
| Ctrl+Shift+H | 隐藏/显示窗口 |
| Ctrl+Shift+C | 复制最新答案 |
| Ctrl+Shift+T | 切换简洁版/展开版 |
| Ctrl+Shift+A | 呼出提问界面 |
| Ctrl+Shift+F | 切换LLM输入模式 |
| Ctrl+Shift+R | 手动触发精修 |
