# 听课助手 (class_copilot)

实时课堂辅助工具，帮助学生在课堂上快速获取高质量的答题参考。

## 功能

- **实时语音转写**：科大讯飞 ASR，支持说话人分离与声纹识别
- **问题检测**：自动识别教师提出的问题
- **智能答案生成**：基于课堂上下文生成口语化参考答案
- **主动问答**：课堂上实时向 AI 提问
- **课堂记录**：保存音频、转写及问答记录

## 快速开始

```bash
# 安装依赖
pip install -e .

# 运行
python -m src.main
```

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+S` | 开始/停止监听 |
| `Ctrl+Shift+Q` | 手动触发提问检测 |
| `Ctrl+Shift+H` | 隐藏/显示悬浮窗 |
| `Ctrl+Shift+C` | 复制最新答案 |
| `Ctrl+Shift+T` | 切换简洁/展开版答案 |
| `Ctrl+Shift+A` | 呼出主动提问输入框 |
| `Ctrl+Shift+F` | 切换 LLM 输入模式 |

## 技术栈

- Python 3.12 + PyQt6
- 科大讯飞实时语音转写（WebSocket）
- 阿里云百炼 Qwen 大模型
- SQLite 本地存储
